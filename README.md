## Overview
This project analyses website traffic and conversion behaviour for BrightCart Online Retail.
The goal was to identify peak engagement periods, evaluate channel effectiveness, and uncover
opportunities to improve conversion and marketing ROI using Advanced Excel.

## Business Problem
BrightCart experienced fluctuating traffic patterns and inconsistent conversion performance.
Limited insight into timing, channel, device, and regional behaviour made it difficult to
schedule campaigns effectively and allocate marketing spend efficiently.

## Objectives
- Identify peak and low engagement periods
- Analyse conversion rates by traffic source
- Compare performance across devices and regions
- Highlight checkout and funnel inefficiencies
- Provide actionable recommendations to improve ROI

## Tools Used
- Microsoft Excel (Advanced Excel)

## Data Scope
- 1,079 user sessions
- Session duration, page views, conversions
- Traffic source, device type, geography
